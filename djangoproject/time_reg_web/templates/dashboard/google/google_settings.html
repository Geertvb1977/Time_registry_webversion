Hoe kom ik aan deze gegevens?

  1. Ga naar de Google Cloud Console en maak een project aan.
  2. Schakel de Google Drive API en Google Docs API in onder APIs & Services.
  3. Configureer je OAuth consent screen (In het consent screen voeg je je eigen beheerder e-mail toe als testgebruiker).
  4. Ga naar Credentials, klik op Create Credentials > OAuth client ID.
  5. Kies Web application en voeg de volgende Redirect URI toe: {{ request.scheme }}://{{ request.get_host }}{% url 'eventaflow:google_callback' %}
  6. Kopieer het verkregen Client ID en Client Secret en plak ze in het formulier hierboven.
